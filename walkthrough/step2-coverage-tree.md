## Coverage Hierarchy

The **Coverage Hierarchy** panel shows the full scope tree from your database:

| Icon | Scope Type |
|------|------------|
| 📦 | Instance / Module |
| 📊 | Covergroup / Coverinstance |
| 📈 | Coverpoint |
| 🔀 | Cross coverage |
| 🔀 | Toggle signal |
| 🌿 | Branch / Condition |
| 🔢 | FSM states / transitions |

Each node shows its **coverage percentage**. Click a node to see detailed bin data.

**Filter buttons** in the panel toolbar let you show only covered, only uncovered, or all items.

> **Tip:** Toggle gutter decorations with `CovSight: Toggle Source Decorations` to see which lines are covered directly in the editor.
